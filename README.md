# Uno Matrix

![RaccoonLabs](https://img.shields.io/badge/Built%20by-RaccoonLabs-blueviolet)

A Matrix/Discord bridge utility for cross-platform communication.

## What It Does
Uno Matrix bridges messages between Matrix rooms and Discord channels, enabling seamless cross-platform conversations. It handles message formatting, user mapping, and media attachments between both protocols.

## Tech Stack
- Python 3.10+
- matrix-nio, discord.py

## Quick Start
```bash
git clone https://github.com/CyberRaccoonTeam/uno-matrix.git
cd uno-matrix
pip install -r requirements.txt
cp config.example.yaml config.yaml
# Edit config.yaml with your tokens
python main.py
```
